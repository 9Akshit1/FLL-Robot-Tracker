Setup Required: Local Agent

Follow these steps to enable port detection and robot communication on this website.

Step 1: Download the file

Click this link to download local_agent.py:

Download local_agent.py

A file named local_agent.py will appear in your Downloads folder.

Step 2: Find your Downloads folder

Look for the local_agent.py file you just downloaded.

Windows: Open File Explorer, look in "Downloads" on the left

Mac: Open Finder, click "Downloads" at the bottom

Linux: Open your file manager, look in Downloads folder

Make sure the file is there before continuing.

Step 3: Open a terminal

Open a command prompt or terminal on your computer:

Windows: Press WIN + R, type cmd, press Enter

Mac: Press CMD + Space, type terminal, press Enter

Linux: Open your terminal application

Step 4: Install required packages

Copy and paste this into your terminal, then press Enter:

pip install flask pyserial mpremote

Wait for the installation to finish (you'll see lines appearing in the terminal).

Step 5: Go to your Downloads folder

In the terminal, type this command and press Enter:

cd Downloads

This moves to your Downloads folder where you saved the file.

Step 6: Run the agent

In the terminal, type this command and press Enter:

python local_agent.py

You should see text like: Starting Flask server on http://0.0.0.0:5001

Keep this terminal open while using this website.

Step 7: Refresh this page

Once you see the message in the terminal, click the button below to refresh this website:

Status: Agent: Offline

If it says "Agent: Online", you're all set! If not, check the terminal for any error messages.
